# Mini Budget Splitter

A small front-end app that helps users split a bill and calculate tip amounts.

## Features

- Enter total bill amount
- Choose number of people
- Add a tip percentage
- Instantly see per-person cost

## Technologies

- HTML
- CSS
- JavaScript

## Purpose

This project was created as a simple utility app and front-end practice project.

## Future Improvements

- Currency selector
- Receipt export
- Local storage support
- Better mobile layout
